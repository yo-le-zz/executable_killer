# executable_killer
Permet de kill tout les processus d'un dossier avec le click droit.
# Pour l'utiliser
Il faut mettre le executable_killer.exe nimporte ou sur votre pc et lancer installer.exe ensuite il va scanner vos disques pour retrouver executable_killer.exe
Une fois qu'il l'aura trouver il va tout installer proprement et vous pourrez utiliser le click droit
Utiliser l'option de desinstallation pour une desinstallation propre vous pouvez voir le fichier executable_killer.exe ici : "C:/tools"
